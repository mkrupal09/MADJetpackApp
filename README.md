# MADJetpackApp

Modern android development with Jetpack Compopse

This App covers following MAD tools, API & Languages

- Fully written in Kotlin
- Jetpack Compose
- Navigation
- LifeCycle
- ViewModel
- Flow
- Hilt (Dependency Injection)
- Coroutines
